双端队列，既可以实现栈，也可以实现队列
```py
import collections
d = collections.deque()

d.append(1) # enqueue -> tail
d.appendleft(2) # enqueue -> head
d.pop() #  dequeue -> tail
d.popleft() # dequeue -> head
```
